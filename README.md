# datepicker-popt
基于社区一个datepicker的项目，增加更改为通过layer-mobile弹出，jquery，更类似安卓手机原生样式

## 下载
<br/>
克隆或者下载安装包到本地解压即可

## 启动
<br/>
在浏览器中打开html文件即可
